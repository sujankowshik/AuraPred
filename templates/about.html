🛠️ Machine Learning Valuation Pipeline

AuraPred uses a state-of-the-art predictive modeling architecture to estimate real estate market valuations. The entire system is divided into two distinct components: the data-preprocessing pipeline and the regression training algorithms.

1. Data Preprocessing & Cleaning

Real estate listing files are often corrupted with human entry anomalies, duplicates, and extreme listings. Our pipeline handles these anomalies recursively before fitting any regressor:

  • Deduplication: The pipeline identifies and purges exact duplicate listings to prevent model overfitting.
  • Imputation of Nulls: Missing property features are filled dynamically: numerical columns (e.g. area size, construction year) are imputed using training medians; categorical columns (e.g. amenities grade) use training modes.
  • Outlier Capping: Extremely large houses or listing prices that skew statistical models are automatically filtered out using the Interquartile Range (IQR) method (capping features outside 1.5 times the middle 50% distribution).
  • Categorical Encoding: Qualitative inputs (Location, Parking, Amenities) are transformed via one-hot encoding using Scikit-Learn’s OneHotEncoder to prevent categorical ordering biases.
  • Feature Scaling: High-variance numerical features (Sq Ft, Year Built, etc.) are standardized using StandardScaler so all inputs contribute harmoniously to optimization thresholds.

2. Trained Models Comparison

We compare multiple machine learning models to identify the most accurate regressor. Our benchmarks evaluated the following algorithms:

Model R² Score (Accuracy) MAE (Avg Error) RMSE (Spread) Strengths / Weaknesses
Gradient Boosting Regressor 98.44% ₹2,116,150 ₹4,028,599 Fits non-linear interaction terms beautifully. Slow training.
Linear Regression 98.22% ₹2,469,259 ₹4,310,188 Ultra-fast, but susceptible to extreme outliers and interactions.
Random Forest Regressor 97.42% ₹3,143,158 ₹5,145,430 Superbly stable ensemble. Tends to overfit very small datasets.

The **Gradient Boosting Regressor** was selected as the primary valuation model due to its high R² accuracy (~98.44%) and low Mean Absolute Error, meaning its pricing predictions deviate by less than 5% of true market prices on average!

🚀 Deployment & Installation

This project is designed to be completely modular and deployment-ready in minutes.

1

Create Environment

Initialize a python virtual environment and run pip installs:

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
2

Train Pipeline

Generate dataset and fit the model ensemble by running:

python3 notebooks/model_training.py
3

Run Local Server

Launch the Flask application locally on port 5001:

python3 app.py
4

Dockerize

Spin up containerized instances using Docker:

docker build -t house-pred .
docker run -p 5001:5001 house-pred
